# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it by opening an issue on the repository. Do not share sensitive or exploitable details in public; instead, provide enough context so maintainers can reproduce and fix the issue. We aim to address any security concerns promptly.
